#### 1. Pull project files from git
If not created, create a folder "projects" or "www" for projects files in WSL, preferably at /home/youruser/ .
Inside folder **projects** or **www** , clone or create git repository of project

#### 2. Get Oracle and SSO files
Create folder "oracle" and "sso" at /home/your_user":![[Pasted image 20250418104728.png]]
Inside "oracle" create folder "instantclient" and place in it instantclient zip files:![[Pasted image 20250418115932.png]]
Files can be found at shared one drive "Programuotojams -> Files".

Inside "sso" folder, place **extracted** content from simplesaml zip file:
![[Pasted image 20250418120120.png]]
And inside sso folder run command to create synlink:
```bash
ln -sfn simplesamlphp-2.0.2 simplesamlphp
```

In project ENV file, edit or add these lines:
```env
SSO_SP=default-sp  
SSO_AUTOLOAD_PATH=/opt/simplesamlphp/src/_autoload.php
```
#### 3. ENV file for MySQL
Create or place .env file at project-name/docker/mysql/.env
Example of file:
```
MYSQL_DATABASE: mokejimai

MYSQL_ROOT_PASSWORD: root

SERVICE_TAGS: dev

SERVICE_NAME: mokejimai_mysql
```


#### 4. Create and run docker container
Go to project folder, and run docker container
```bash
docker compose up -d
```
For first time, it may take a few minutes to build images for web application. 
#### 5. Project configuration and database
Create value at windows hosts file(C:\Windows\System32\drivers\etc\hosts):![[Pasted image 20250331113436.png]]
In the project configuration file, ensure that the database host matches the container name specified in the Docker configuration file for the database image.
For importing database, use navicat and connect to localhost, just use respective port from [[Projects port mappings]]
#### 6. Project start
Try to open web application in browser. You should also specify the appropriate port from the [port mapping table](obsidian://open?vault=General&file=WSL%20%2B%20DOCKER%2FProjects%20port%20mappings) in URL: ![[Pasted image 20250331113855.png]]
