
#### 1. Install WSL
In terminal run:
```sh
wsl --install
```
It will install WSL with the latest Ubuntu LTS and **RESTART** PC
#### 2. Start WSL
Start Ubuntu via windows search and configure it
![[Pasted image 20250220133945.png]]
#### 3. Allocating more resources for WSL
##### 1. Shut down WSL 
```sh
wsl --shutdown
```
##### 2. Create configuration file
Create a configuration file **.wslconfig** at your user profile directory:
C:\Users\59089 (use your own id)
##### 3. Add parameters to configuration file
The following parameters define their corresponding resources:
	_CPU_ = > cores dedicated to WSL
	_Memory_ = > RAM allocated to WSL
	_AutoMount_ = > Mount your default NTFS drive to your subsystem
	_Boot_ => defines your booting options for services
	_Kernel_ = > Custom kernel for linux
	_LocalhostForwarding_ = > for port-forwarding your applications running on localhost
